## Commandes

Voici la liste des commandes proposées par Denver :

| Commande | Description                                      | Permissions |
|----------|--------------------------------------------------|-------------|
| ping     | Répond pong                                      | User        |

| invite   | Affiche le lien d'invitation pour ajouter Denver | User        |
